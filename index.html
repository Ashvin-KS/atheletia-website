FAQ

Questions.

Is Atheletia really free?

+

Yes. Completely free and open source under the MIT License. You only provide your own API keys for AI features.

Does my data leave my machine?

+

Absolutely. All data is stored locally in SQLite on your device. API keys go to your OS keyring. Nothing is collected.

Which AI models does it support?

+

OpenAI, NVIDIA NIM, Anthropic, Groq, Gemini, and local LM Studio endpoints. Switch anytime in Settings.

Will it slow down my PC?

+

Not at all. Atheletia is a native binary built with Rust. It uses less RAM than a single browser tab.

Can I use it without any AI provider?

+

Absolutely. The scheduler, music player, LeetCode browser, activity tracker, vault, and Pomodoro timer all work completely offline. AI features degrade gracefully when no key is configured.

How do I install it?

+

Download the Windows .msi installer from GitHub and run it. Requires Windows 10 or later.
